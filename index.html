Introduction to CSS Floats

Float Left

Floats are a powerful CSS property that allows elements to be positioned to the left or right of their container, with other content flowing around them. This technique revolutionized web layout design and remains relevant today.

Basic Float Properties


.float-left {
    float: left;
}

.float-right {
    float: right;
}

.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
